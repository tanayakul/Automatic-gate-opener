# Automatic Gate Opener Using Arduino UNO

## Project Overview
The Automatic Gate Opener is an IoT-based automation project designed to open and close a gate automatically when a vehicle or person approaches. The system uses an Arduino UNO, an ultrasonic sensor, and a servo motor to detect objects and control gate movement without manual intervention.

## Objectives

1. Design an automated gate system.
2. Reduce manual effort in gate operation.
3. Improve convenience and security.
4. Demonstrate the use of sensors and microcontrollers in automation.
5. Explore IoT applications in smart homes and smart cities.

## Components Used

1. Arduino UNO
2. HC-SR04 Ultrasonic Sensor
3. Servo Motor
4. Breadboard
5. Jumper Wires
6. Power Supply

## Working Principle
The ultrasonic sensor continuously measures the distance of nearby objects. When an object is detected within a predefined range, the Arduino UNO processes the sensor data and commands the servo motor to open the gate. After a short delay, the servo motor returns to its original position, closing the gate automatically.


## Circuit Connections

1. Trigger Pin → Arduino Pin 9
2. Echo Pin → Arduino Pin 10
3. Servo Signal Pin → Arduino Pin 11
4. VCC and GND connected appropriately
